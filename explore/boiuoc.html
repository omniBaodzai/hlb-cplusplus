1. Ý tưởng cơ bản

Giả sử bạn có số nguyên dương n.

👉 Một số i là ước của n nếu:

n % i == 0

Cách đơn giản nhất

  • Duyệt tất cả các số từ 1 → n

  • Nếu i chia hết n thì in ra

📌 Ví dụ: n = 12
Các ước là: 1, 2, 3, 4, 6, 12

2. Hiện thực C++ (cách cơ bản)

C++
#include <iostream>
using namespace std;

int main() {
    int n;
    cout << "Nhap n: ";
    cin >> n;

    cout << "Cac uoc cua " << n << " la: ";

    for (int i = 1; i <= n; i++) {
        if (n % i == 0) {
            cout << i << " ";
        }
    }

    return 0;
}

3. Nhận xét vấn đề

Cách trên chạy O(n) → chậm nếu n lớn (ví dụ n = 10^9)

4. Ý tưởng tối ưu (rất quan trọng)

Nếu i là ước của n thì:

n / i cũng là ước

👉 Các ước luôn xuất hiện theo cặp

Ví dụ:

  • 12 → (1,12), (2,6), (3,4)

📌 Vì vậy:

  • Chỉ cần duyệt từ 1 → sqrt(n)

5. Hiện thực tối ưu C++

C++
#include <iostream>
#include <cmath>
using namespace std;

int main() {
    int n;
    cout << "Nhap n: ";
    cin >> n;

    cout << "Cac uoc cua " << n << " la: ";

    for (int i = 1; i <= sqrt(n); i++) {
        if (n % i == 0) {
            cout << i << " ";

            if (i != n / i) { // tránh in trùng
                cout << n / i << " ";
            }
        }
    }

    return 0;
}

6. Sắp xếp các ước (nâng cao)

Cách tối ưu ở trên sẽ in không theo thứ tự

👉 Nếu muốn tăng dần:

  • Lưu vào mảng/vector

  • Sort lại

C++
#include <iostream>
#include <vector>
#include <algorithm>
#include <cmath>
using namespace std;

int main() {
    int n;
    cin >> n;

    vector<int> uoc;

    for (int i = 1; i <= sqrt(n); i++) {
        if (n % i == 0) {
            uoc.push_back(i);
            if (i != n / i) {
                uoc.push_back(n / i);
            }
        }
    }

    sort(uoc.begin(), uoc.end());

    for (int x : uoc) {
        cout << x << " ";
    }

    return 0;
}

  • Cách 2: Duyệt đến sqrt(n) → nhanh hơn rất nhiều

  • Mấu chốt:

    Ước luôn đi theo cặp

Khi chuyển từ tìm ước của 1 số sang 2 số a và b, bài toán thường có 2 hướng phổ biến:

1. Bài toán 1: Tìm ước chung của a và b

🔹 Ý tưởng

Một số i là ước chung của a và b nếu:

C++
a % i == 0 && b % i == 0

👉 Nghĩa là:

  • i phải chia hết cả a và b

🔹 Cách đơn giản

  • Duyệt từ 1 → min(a, b)

  • Kiểm tra điều kiện

🔹 C++ (cách cơ bản)

C++
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;

    cout << "Cac uoc chung: ";

    for (int i = 1; i <= min(a, b); i++) {
        if (a % i == 0 && b % i == 0) {
            cout << i << " ";
        }
    }

    return 0;
}

🔹 Nhận xét

  • Độ phức tạp: O(min(a, b)) → chậm nếu số lớn

2. Tối ưu bằng tư duy quan trọng

👉 Nhận xét:

Ước chung của a và b chính là ước của GCD(a, b)

🔹 Ý tưởng mới

  1. Tìm:

C++
g = gcd(a, b)

  1. Tìm các ước của g

🔹 Ví dụ

a = 12, b = 18
→ gcd = 6
→ Ước chung: 1, 2, 3, 6

🔹 C++ (tối ưu)

C++
#include <iostream>
#include <cmath>
using namespace std;

// Hàm tìm GCD
int gcd(int a, int b) {
    while (b != 0) {
        int r = a % b;
        a = b;
        b = r;
    }
    return a;
}

int main() {
    int a, b;
    cin >> a >> b;

    int g = gcd(a, b);

    cout << "Cac uoc chung: ";

    for (int i = 1; i <= sqrt(g); i++) {
        if (g % i == 0) {
            cout << i << " ";

            if (i != g / i) {
                cout << g / i << " ";
            }
        }
    }

    return 0;
}

3. Bài toán 2: Tìm ước chung lớn nhất (GCD)

🔹 Ý tưởng (Euclid)

C++
gcd(a, b) = gcd(b, a % b)

Lặp đến khi b = 0

🔹 C++ đơn giản

C++
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;

    while (b != 0) {
        int r = a % b;
        a = b;
        b = r;
    }

    cout << "GCD = " << a;

    return 0;
}

4. Bài toán 3 (mở rộng): Ước chung nhỏ nhất (LCM)

🔹 Công thức

C++
lcm(a, b) = (a * b) / gcd(a, b)

🔹 C++

C++
#include <iostream>
using namespace std;

int gcd(int a, int b) {
    while (b != 0) {
        int r = a % b;
        a = b;
        b = r;
    }
    return a;
}

int main() {
    int a, b;
    cin >> a >> b;

    int g = gcd(a, b);
    int lcm = (a * b) / g;

    cout << "LCM = " << lcm;

    return 0;
}

5. Tóm tắt tư duy quan trọng

👉 Có 3 mức:

  1. Ngây thơ

    • Duyệt từ 1 → min(a, b)

  2. Tối ưu

    • Tìm gcd(a, b)

    • Tìm ước của gcd

  3. Cốt lõi toán học

    • Ước chung = Ước của GCD

    • GCD dùng thuật toán Euclid

6. Chương trình hoàn chỉnh

#include <iostream>
#include <vector>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;

    vector<int> uocA, uocB;

    // Ước của a

        if (a % i == 0) {
            uocA.push_back(i);
        }

    for (int i = 1; i <= b; i++) {
        if (b % i == 0) {
            uocB.push_back(i);
        }
    }

    // In ước của a
    cout << "Uoc cua a: ";
    for (int x : uocA) {
        cout << x << " ";
    }

    for (int x : uocB) {
        cout << x << " ";
    }

    // Tìm và in ước chung
    cout << "\nUoc chung: ";
    for (int i = 0; i < uocA.size(); i++) {
        for (int j = 0; j < uocB.size(); j++) {
            if (uocA[i] == uocB[j]) {
                cout << uocA[i] << " ";
            }
        }
    }

}

ChatGPT đã nói:

Bài toán tìm ước chung lớn nhất (GCD – Greatest Common Divisor) là một trong những nền tảng quan trọng nhất trong lập trình và toán học. Mình sẽ giải thích theo đúng kiểu “từng bước tư duy → code”.

1. Ước chung lớn nhất là gì?

👉 GCD của 2 số a và b là:

Số lớn nhất chia hết cho cả a và b

📌 Ví dụ:

  • a = 12, b = 18
    → Ước của 12: 1, 2, 3, 4, 6, 12
    → Ước của 18: 1, 2, 3, 6, 9, 18
    → Ước chung: 1, 2, 3, 6
    👉 GCD = 6

2. Cách 1: Ý tưởng cơ bản (dễ hiểu)

🔹 Ý tưởng

  • Duyệt từ 1 → min(a, b)

  • Nếu số nào chia hết cả 2 thì:

    • Cập nhật kết quả

🔹 Code C++

C++
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;

    int gcd = 1;

    for (int i = 1; i <= min(a, b); i++) {
        if (a % i == 0 && b % i == 0) {
            gcd = i;
        }
    }

    cout << "GCD = " << gcd;

    return 0;
}

🔹 Nhận xét

  • Dễ hiểu

  • Nhưng chậm: O(min(a, b))

3. Cách 2: Thuật toán Euclid (QUAN TRỌNG)

Đây là cách chuẩn, nhanh và rất hay ra thi.

🔹 Ý tưởng cốt lõi

👉 Quy tắc:

C++
gcd(a, b) = gcd(b, a % b)

Lặp lại cho đến khi:

C++
b == 0

👉 Khi đó:

C++
gcd = a

🔹 Ví dụ chi tiết

Tìm GCD(12, 18):

aba % b
121812
18126
1260

👉 Kết quả: 6

6. Cách 3: Dùng đệ quy (hiểu sâu hơn)

🔹 Ý tưởng

Áp dụng trực tiếp công thức:

C++
gcd(a, b) = gcd(b, a % b)

🔹 Code

C++
#include <iostream>
using namespace std;

int gcd(int a, int b) {
    if (b == 0) return a;
    return gcd(b, a % b);
}

int main() {
    int a, b;
    cin >> a >> b;

    cout << "GCD = " << gcd(a, b);

    return 0;
}

ChatGPT đã nói:

Khi chuyển từ 2 số → 3 số trở lên, điều quan trọng là bạn không cần phát minh thuật toán mới, mà chỉ cần mở rộng từ GCD của 2 số.

1. Ý tưởng cốt lõi

👉 Với 3 số a, b, c:

C++
gcd(a, b, c) = gcd(gcd(a, b), c)

👉 Với nhiều số hơn:

C++
gcd(a, b, c, d, ...) = gcd(gcd(gcd(a, b), c), d)...

📌 Tức là:

“Gộp từng cặp lại”

2. Ví dụ cụ thể

Tìm GCD của: 12, 18, 24

  • Bước 1: gcd(12, 18) = 6

  • Bước 2: gcd(6, 24) = 6

👉 Kết quả: 6

3. Xây dựng từng bước

🔹 Bước 1: Viết hàm GCD cho 2 số (Euclid)

C++
int gcd(int a, int b) {
    while (b != 0) {
        int r = a % b;
        a = b;
        b = r;
    }
    return a;
}

🔹 Bước 2: Mở rộng cho 3 số

👉 Ý tưởng:

  • Tính gcd(a, b)

  • Sau đó lấy kết quả đi tính tiếp với c

🔹 Code

C++
#include <iostream>
using namespace std;

int gcd(int a, int b) {
    while (b != 0) {
        int r = a % b;
        a = b;
        b = r;
    }
    return a;
}

int main() {
    int a, b, c;
    cin >> a >> b >> c;

    int result = gcd(a, b);
    result = gcd(result, c);

    cout << "GCD = " << result;

    return 0;
}

4. Tổng quát cho n số

🔹 Ý tưởng

  • Nhập n số

  • Gán:

C++
result = số đầu tiên

  • Sau đó:

C++
result = gcd(result, số tiếp theo)

🔹 Code C++

C++
#include <iostream>
using namespace std;

int gcd(int a, int b) {
    while (b != 0) {
        int r = a % b;
        a = b;
        b = r;
    }
    return a;
}

int main() {
    int n;
    cin >> n;

    int x;
    cin >> x;

    int result = x;

    for (int i = 1; i < n; i++) {
        cin >> x;
        result = gcd(result, x);
    }

    cout << "GCD = " << result;

    return 0;
}

🔥 Bội của LCM (bội chung nhỏ nhất)

🔹 Công thức mở rộng

Với 3 số:

C++
lcm(a, b, c) = lcm(lcm(a, b), c)

Với n số:

C++
lcm(a, b, c, d, ...) = lcm(lcm(lcm(a, b), c), d)...

3. Ví dụ cụ thể

Tìm bội chung của: 4, 6, 8

  • Bước 1: lcm(4, 6) = 12

  • Bước 2: lcm(12, 8) = 24

👉 Bội chung:

24, 48, 72, 96, ...

4. Xây dựng từng bước

🔹 Bước 1: Viết hàm GCD (Euclid)

C++
int gcd(int a, int b) {
    while (b != 0) {
        int r = a % b;
        a = b;
        b = r;
    }
    return a;
}

🔹 Bước 2: Viết hàm LCM

C++
int lcm(int a, int b) {
    return (a * b) / gcd(a, b);
}

🔹 Bước 3: Tính LCM của nhiều số

👉 Ý tưởng:

  • Gán:

C++
result = số đầu tiên

  • Lặp:

C++
result = lcm(result, số tiếp theo)

🔹 Code

C++
#include <iostream>
using namespace std;

int gcd(int a, int b) {
    while (b != 0) {
        int r = a % b;
        a = b;
        b = r;
    }
    return a;
}

int lcm(int a, int b) {
    return (a * b) / gcd(a, b);
}

int main() {
    int n;
    cin >> n;

    int x;
    cin >> x;

    int result = x;

    for (int i = 1; i < n; i++) {
        cin >> x;
        result = lcm(result, x);
    }

    cout << "LCM = " << result << endl;

5. Chương trình hoàn chỉnh

#include <iostream>

int gcd(int a, int b) {
    while (b != 0) {
        int r = a % b;
        a = b;
        b = r;
    }
    return a;
}

int lcm(int a, int b) {
    return (a * b) / gcd(a, b);

int main() {
    int n;

    int result = x;

“Bội chung nhỏ nhất” chính là tên gọi chuẩn của LCM (Least Common Multiple). Đây là bài rất quan trọng vì nó thường đi kèm với GCD.

Mình sẽ hướng dẫn theo đúng kiểu: ý tưởng → trực giác → từng bước code C++ → tối ưu.

1. Bội chung nhỏ nhất là gì?

👉 Với 2 số a và b, bội chung nhỏ nhất (LCM) là:

Số dương nhỏ nhất chia hết cho cả a và b

🔹 Ví dụ

a = 4, b = 6

  • Bội của 4: 4, 8, 12, 16, ...

  • Bội của 6: 6, 12, 18, ...

👉 Bội chung nhỏ nhất = 12

2. Cách 1: Làm theo định nghĩa (dễ hiểu nhất)

🔹 Ý tưởng

  • Bắt đầu từ số lớn hơn giữa a và b

  • Tăng dần từng bước

  • Kiểm tra số nào chia hết cả 2

🔹 Code C++

C++
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;

    int lcm = max(a, b);

    while (true) {
        if (lcm % a == 0 && lcm % b == 0) {
            break;
        }
        lcm++;
    }

    cout << "LCM = " << lcm;

    return 0;
}

🔹 Nhận xét

  • Dễ hiểu

  • ❌ Rất chậm nếu số lớn

3. Ý tưởng quan trọng để tối ưu

👉 Nhận xét toán học cực quan trọng:

LCM và GCD có liên hệ:

C++
LCM(a, b) * GCD(a, b) = a * b

👉 Suy ra:

C++
LCM(a, b) = (a * b) / GCD(a, b)

4. Bước 1: Tìm GCD (Euclid)

🔹 Ý tưởng GCD

C++
gcd(a, b) = gcd(b, a % b)

🔹 Code GCD

C++
int gcd(int a, int b) {
    while (b != 0) {
        int r = a % b;
        a = b;
        b = r;
    }
    return a;
}

5. Bước 2: Tính LCM

C++
(a * b) / gcd(a, b)

🔹 Code hoàn chỉnh tối ưu

C++
#include <iostream>
using namespace std;

int gcd(int a, int b) {
    while (b != 0) {
        int r = a % b;
        a = b;
        b = r;
    }
    return a;
}

int main() {
    int a, b;
    cin >> a >> b;

    long long lcm = (1LL * a * b) / gcd(a, b);

    cout << "LCM = " << lcm;

    return 0;
}

6. Vì sao công thức đúng? (hiểu bản chất)

👉 Ý tưởng:

  • GCD = phần chung bị “trùng”

  • Khi nhân a × b → bị đếm trùng phần GCD

  • Chia lại cho GCD để loại trùng

7. Ví dụ minh họa

a = 12, b = 18

  • gcd = 6

  • a × b = 216

  • 216 / 6 = 36

👉 LCM = 36
