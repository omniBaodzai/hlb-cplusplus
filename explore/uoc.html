Cách cơ bản, dễ hiểu nhất để tìm Ước chung lớn nhất (GCD) là:

Ý tưởng (cách 1 – duyệt từ nhỏ đến lớn)

  • Duyệt tất cả các số từ 1 đến min(a, b)

  • Nếu số đó chia hết cho cả a và b thì là ước chung

  • Lưu lại ước chung lớn nhất

Code C++ (cơ bản, dễ hiểu)

C++
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;

    int gcd = 1; // giả sử nhỏ nhất là 1

    for (int i = 1; i <= min(a, b); i++) {
        if (a % i == 0 && b % i == 0) {
            gcd = i;
        }
    }

    cout << gcd;
    return 0;
}

Ví dụ

Input:

12 18

Các ước chung: 1, 2, 3, 6
→ GCD = 6

🧠 Ý tưởng Euclid (cốt lõi)

Thay vì duyệt từ 1 → min(a,b), ta dùng tính chất:

GCD(a, b) = GCD(b, a % b)

Lặp lại cho đến khi b = 0 thì a chính là GCD.

🔁 Cách hoạt động từng bước

Ví dụ: a = 12, b = 18

Bước 1:

GCD(12, 18)
→ đổi thành GCD(18, 12 % 18)
→ 12 % 18 = 12
→ GCD(18, 12)

Bước 2:

GCD(18, 12)
→ GCD(12, 18 % 12)
→ 18 % 12 = 6
→ GCD(12, 6)

Bước 3:

GCD(12, 6)
→ GCD(6, 12 % 6)
→ 12 % 6 = 0
→ GCD(6, 0)

Kết thúc:

GCD(6, 0) = 6

👉 Kết quả: 6

💻 Code C++ (Euclid – chuẩn CP)

C++
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    long long a, b;
    cin >> a >> b;

    while (b != 0) {
        long long r = a % b;
        a = b;
        b = r;
    }

    cout << a;
    return 0;
}

Mỗi vòng lặp:

👉 log(N) vòng lặp thay vì O(N)

CáchĐộ phức tạpÝ tưởng
DuyệtO(n)thử từng số

💻 Code C++ (đệ quy)

C++
#include <bits/stdc++.h>
using namespace std;

long long gcd(long long a, long long b) {
    if (b == 0) return a;
    return gcd(b, a % b);
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    long long a, b;
    cin >> a >> b;

    cout << gcd(a, b);
    return 0;
}

🧠 Cách chạy (ví dụ 12, 18)

→ gcd(18, 12)
→ gcd(12, 6)
→ gcd(6, 0)

💻 Code C++ (cách phân tích thừa số)

C++
#include <bits/stdc++.h>
using namespace std;

map<long long, int> factorize(long long n) {
    map<long long, int> f;

    for (long long i = 2; i * i <= n; i++) {
        while (n % i == 0) {
            f[i]++;
            n /= i;
        }
    }

    if (n > 1) f[n]++;

    return f;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    long long a, b;
    cin >> a >> b;

    auto fa = factorize(a);
    auto fb = factorize(b);

    long long gcd = 1;

    for (auto &p : fa) {
        long long prime = p.first;
        int expA = p.second;

        if (fb.count(prime)) {
            int expB = fb[prime];
            int minExp = min(expA, expB);

            while (minExp--) {
                gcd *= prime;
            }
        }
    }

    cout << gcd;
    return 0;
}
