1️⃣ islower() – kiểm tra ký tự thường

Kiểm tra xem ký tự có phải chữ cái thường a → z không.

Ví dụ

C++
#include <iostream>
#include <cctype>
using namespace std;

int main()
{
    char c = 'g';

    if (islower(c))
        cout << "Ky tu thuong";
    else
        cout << "Khong phai ky tu thuong";
}

✔ 'g' → true
❌ 'G' → false
❌ '5' → false

2️⃣ isupper() – kiểm tra ký tự hoa

Kiểm tra ký tự có phải A → Z.

Ví dụ

C++
char c = 'A';

if (isupper(c))
    cout << "Ky tu hoa";

✔ 'A' → true
❌ 'a' → false

3️⃣ isdigit() – kiểm tra ký tự số

Kiểm tra ký tự có phải '0' → '9'.

Ví dụ

C++
char c = '8';

if (isdigit(c))
    cout << "Day la ky tu so";

✔ '5' → true
❌ 'a' → false

⚠️ Lưu ý:

isdigit('5') = true
isdigit(5) = false

vì '5' là ký tự, còn 5 là số nguyên.

4️⃣ tolower() – chuyển về chữ thường

Chuyển chữ hoa → chữ thường.

Ví dụ

C++
char c = 'A';

c = tolower(c);

cout << c;

Output:

5️⃣ toupper() – chuyển về chữ hoa

Chuyển chữ thường → chữ hoa.

C++
char c = 'b';

c = toupper(c);

cout << c;

Output:
